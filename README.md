#### tapi宗旨
打造高效、易用、功能强大的API管理平台，旨在为开发、产品、测试人员提供更优雅的接口管理服务

#### 技术栈
laravel+vue+elementUI  

#### 结构
> tapi-plugin 为chrome扩展， 为本地安装

> tapi-sev 为服务端代码， 文档参考该文件夹下readme

> tapi-web 为前端代码， 文档参考该文件夹下readme
